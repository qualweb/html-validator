# html-validator
W3C html5 markup validator

## How to install

```shell
  $ git clone https://github.com/qualweb/html-validator.git
  $ cd html-validator
  $ npm install
```

### Dependencies

Java must be installed in the system.

## How to run

```shell
  $ npm run start
```

## Webservice

http://localhost:5555/url/:url

Replace the :url for the desired url to validate.
